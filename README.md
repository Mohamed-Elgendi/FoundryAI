```
├── Box breathing: 4 counts (in, hold, out, hold)
├── 5-4-3-2-1 senses technique
└── Physical grounding: feet on floor, back straight

STEP 2: Micro-Task Entry (5 minutes)
├── AI selects: "Easiest possible task related to goal"
├── Example: "Open the editor" (not "Build the feature")
├── Complete with full celebration
└── Confidence restoration

STEP 3: Gradual Expansion
├── Next slightly harder task
├── Maintain manageable challenge
└── → Natural flow state emergence
```

**Protocol: Discouragement → Flow**

```
DETECTED: Comparison language, self-doubt, "I can't" statements

STEP 1: Evidence Review (3 minutes)
├── AI surfaces: "Remember when you..."
├── Dashboard: Past completed tasks, streaks, wins
├── Community: "3 people learned from your post last week"
└── Objective evidence of capability

STEP 2: Self-Compassion Exercise (2 minutes)
├── "What would you say to a friend feeling this way?"
├── Reframe: "This is hard. Everyone struggles. I'm learning."
├── Permission to be where you are
└── Emotional safety restoration

STEP 3: Tiny Step
├── Smallest meaningful action
├── "Write one function" not "Build the app"
├── Completion = proof of progress
└── → Momentum builds naturally
```

**Protocol: Overwhelm → Flow**

```
DETECTED: Multiple open tasks, avoidance, scattered attention

STEP 1: Emergency Brain Dump (5 minutes)
├── Everything on mind → captured
├── AI sorts: Delete/Delegate/Do
└── Mental RAM cleared

STEP 2: Single Focus Selection
├── AI recommendation: "Given your state, this ONE task"
├── User override: "Actually, I want to do X"
├── Full commitment to single thread
└── Multi-tasking eliminated

STEP 3: Time Boxing
├── "Just 25 minutes on this one thing"
├── Timer visible, break promised
├── Permission to stop after (usually continues)
└── → Flow through manageable commitment
```

## 📊 Database Schema

```sql
-- User journeys and progress tracking
create table public.user_journeys (
  id uuid default gen_random_uuid() primary key,
  user_id uuid references auth.users(id),
  current_stage text not null,
  selected_idea jsonb,
  build_progress integer default 0,
  revenue_generated decimal(10,2) default 0,
  milestones_achieved text[] default '{}',
  created_at timestamp with time zone default timezone('utc'::text, now()),
  updated_at timestamp with time zone default timezone('utc'::text, now())
);

-- AI-generated opportunities from radar
create table public.opportunities (
  id uuid default gen_random_uuid() primary key,
  title text not null,
  description text,
  category text,
  difficulty_level text,
  monetization_potential integer,
  competition_level text,
  trending_score decimal(5,2),
  keywords text[],
  template_data jsonb,
  is_active boolean default true,
  created_at timestamp with time zone default timezone('utc'::text, now())
);

-- Template gallery
create table public.templates (
  id uuid default gen_random_uuid() primary key,
  name text not null,
  description text,
  category text,
  mvp_features text[],
  tech_stack jsonb,
  build_roadmap jsonb,
  monetization_strategy jsonb,
  estimated_build_days integer,
  difficulty text,
  downloads_count integer default 0,
  success_rate decimal(5,2),
  is_featured boolean default false
);

-- User feedback for AI improvement
create table public.feedback (
  id uuid default gen_random_uuid() primary key,
  user_input text not null,
  output_json jsonb not null,
  is_helpful boolean not null,
  user_agent text,
  created_at timestamp with time zone default timezone('utc'::text, now())
);
```
