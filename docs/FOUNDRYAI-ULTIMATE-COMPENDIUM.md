```typescript
    models: ['meta-llama/llama-3-70b', 'anthropic/claude-3-opus'],

    maxTokens: 128000,

  }

```

**Failover Flow:**
1. Try Primary (Groq) → Success? Return response
2. Fail? Try Secondary (OpenRouter) → Success? Return response
3. Fail? Check cache → Hit? Return cached
4. Miss? Queue for background processing

**Circuit Breaker:**
- Monitor failure rate per provider
- >10% failures in 1 min = Open circuit
- Route to healthy providers
- After 30s = Half-open (test health)
- Healthy? Close circuit, resume normal

**Caching Strategy:**
- L1: In-Memory (5-60s TTL)
- L2: Redis (AI: 1hr, Opportunities: 5min, Sessions: 24hr)
- L3: CDN (Static: 1 year, API: varies)
