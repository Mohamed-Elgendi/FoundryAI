| **Comparison Engines** | Side-by-side product comparisons | Multiple affiliate links | Medium |

## Archetype 6: PaaS (Platform as a Service)

### 🏪 Multi-Sided Marketplaces

**The Model:** Building platforms that connect buyers and sellers, earning through transaction fees and premium features.

#### Platform Types

| Platform Type | Two Sides | Revenue Model | Examples |
|--------------|-----------|---------------|----------|
| **Job Boards** | Employers + Job seekers | Listing fees, featured jobs | We Work Remotely, AngelList |
| **Freelance Marketplaces** | Clients + Freelancers | Transaction fees (5-20%) | Upwork, Fiverr |
| **Product Marketplaces** | Buyers + Sellers | Transaction fees, listing fees | Etsy, Gumroad |
| **Service Directories** | Service seekers + Providers | Lead fees, subscriptions | Thumbtack, Yelp |
| **Rental Platforms** | Renters + Owners | Booking fees, service fees | Airbnb, Turo |
| **Community Platforms** | Members + Content creators | Membership fees, tips | Patreon, Substack |

#### Build Strategy

```
┌─────────────────────────────────────────────────────────────────┐
│  PAAS PLATFORM DEVELOPMENT STRATEGY                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                   │
│  THE CHICKEN-AND-EGG SOLUTION:                                    │
│  ├── Phase 1: Supply-First (Critical)                             │
│  │   • Manually recruit initial sellers/providers               │
│  │   • Offer incentives: Free listings, revenue guarantees      │
│  │   • Quality over quantity: Curate 10-50 excellent providers  │
│  │   • Provide value: Help them succeed (not just list them)    │
│  │                                                               │
│  ├── Phase 2: Demand Generation                                   │
│  │   • Targeted marketing to buyers/seekers                     │
│  │   • Content marketing: "Find the best [providers]"           │
│  │   • SEO: Capture high-intent search traffic                  │
│  │   • Partnerships: Integrations with complementary tools        │
│  │                                                               │
│  └── Phase 3: Network Effects Activation                          │
│      • Self-sustaining growth: More sellers → More buyers       │
│      • → More sellers (cycle continues)                           │
│      • Platform improvements: Better matching, trust systems    │
│                                                                   │
│  TRUST & SAFETY SYSTEMS:                                          │
│  ├── Verification: Identity, credentials, portfolio review       │
│  ├── Reviews: Two-sided rating system                            │
│  ├── Dispute resolution: Clear process, mediation               │
│  ├── Insurance/Guarantees: Platform backing for transactions    │
│  └── Quality control: Manual review, algorithmic detection      │
│                                                                   │
└─────────────────────────────────────────────────────────────────┘
```

#### 4-Pillar Success System

**Go-to-Market:**
- Seed both sides → Focus on supply first → Demand generation
- Geographic/vertical expansion: Dominate one niche, then expand
- Quality curation: Be known for the best in category

**Sales Funnel:**
