## Getting Started

### Prerequisites

- **Node.js 18+**
- **Git** installed
- **Free accounts** for essential development services (AI will guide you based on your tier)

### Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/Mohamed-Elgendi/FoundryAI.git
cd FoundryAI

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp env.example .env.local

# 4. Edit .env.local with your keys
# NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
# NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
# GROQ_API_KEY=your_groq_key
# OPENROUTER_API_KEY=your_openrouter_key (optional, for fallback)

# 5. Set up database
# Run supabase/schema.sql in your Supabase SQL editor

# 6. Start development server
npm run dev

# 7. Open http://localhost:3000
```
