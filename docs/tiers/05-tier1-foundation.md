**Core Principle:** *A cluttered mind cannot create. Clear the cache, unlock capacity.*

### The Brain Dump Protocol

```
┌─────────────────────────────────────────────────────────────────┐
│  THE BRAIN DUMP PROTOCOL                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                   │
│  PRE-SESSION RITUAL (3-5 minutes):                              │
│                                                                   │
│  STEP 1: CAPTURE (2 minutes) - Write everything               │
│  ├── Open tasks lingering in mind                               │
│  ├── Worries or concerns (personal and professional)            │

│  └── ANYTHING occupying mental space                            │
│  → Goal: Empty brain onto paper/screen                          │
│                                                                   │
│  STEP 2: CATEGORIZE (1 minute) - Sort into buckets              │
│  ├── URGENT: Needs immediate attention (handle now or schedule) │
│  ├── IMPORTANT: Must do today (add to today's list)           │
│  ├── LATER: Can wait (schedule for future date)               │
│  ├── DELEGATE: Someone else can do (assign or request)          │

│  ├── "These items are safely stored"                            │
│  ├── "I will address them at their scheduled time"            │

│  └── Close the Brain Dump document/app                        │
│                                                                   │
│  STEP 4: CENTER (30 seconds) - Transition to work             │
│  ├── Deep breath                                                │
│  ├── Set session intention                                      │
│  ├── Begin deep work with clear mind                          │
│  └── Full cognitive resources available for creation            │
│                                                                   │
└─────────────────────────────────────────────────────────────────┘
```
