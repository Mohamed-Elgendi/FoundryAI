## AI Router: Multi-Provider Failover

```typescript
// Primary: Groq (70% of traffic)
// - 10x faster inference
// - Cost-effective (Mixtral, Llama 3)
// - High rate limits (1000 req/min)

// Secondary: OpenRouter (25% of traffic)
// - Access to 100+ models
// - Claude 3 Opus for complex tasks

// Fallback: Queue (5% of traffic)
// - Background processing
// - <5 minute SLA
```

**Failover Logic:**
1. Try Primary (Groq)
2. If fails (< 5 sec) → Try Secondary (OpenRouter)
3. If fails → Serve from cache or queue for retry
